Setup Guide

  1. Install the GSI Config

    Click Install CS2 GSI Config on the Status tab. This writes a small config file to your CS2 folder so the game sends health updates to RespawnTok.

  2. Use Fullscreen Windowed mode in CS2

    Go to CS2 → Settings → Video → Display Mode and set it to Fullscreen Windowed. This lets RespawnTok bring its window to the front without minimising the game.

  3. Launch CS2 and play

    When your HP reaches 0 the TikTok feed appears automatically. When you respawn (HP > 0) it hides itself.

  4. Manual dismiss

    Press Alt + X to close the viewer early (e.g. if you respawn before it hides).

  5. TikTok Login (optional)

    The TikTok webview loads tiktok.com/foryou. You can log in inside the viewer window and your session will persist between launches.
