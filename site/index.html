How It Works

Three steps. Zero effort.

01
⚙️

Install the Config

Click one button. RespawnTok writes the Game State Integration file directly into your CS2 or Dota 2 folder.

02
🎮

Play Normally

Use Fullscreen Windowed mode. RespawnTok runs silently in the background — no performance impact, no overlays.

03
📱

Die → TikTok

The moment your HP hits zero, your TikTok For You feed fades in. Respawn and it vanishes. Alt+X to dismiss early.

Features

Built for gamers.

🔫

CS2 Support

Full Game State Integration for Counter-Strike 2. Tracks your exact health transition — not just round start.

🗡️

Dota 2 Support

Uses hero.alive from Dota 2 GSI — fires instantly when you fall, clears when you respawn.

⚡

Instant Fade

Smooth 200ms fade-in on death, fade-out on respawn. No jarring snaps. Your teammates won't even notice.

🖥️

Anti-Minimize

Works in Fullscreen Windowed mode so CS2 stays live behind the overlay. No alt-tab penalty.

⌨️

Global Hotkey

Press Alt+X anywhere to instantly dismiss the viewer — or toggle pause mode if you're spectating.

🧪

Test Mode

No live game needed. Simulate death and respawn events with one click to verify your setup before queuing.

🛡️

VAC Safe

Uses Valve's official Game State Integration API only. Reads data; never writes to game memory. No ban risk.

🪟

Standalone

No Overwolf. No subscriptions. A single .exe installer — runs on any Windows 10/11 machine.
